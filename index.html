Idea de proyecto: Billetera Virtual

Pasos a seguir para ejecutar la billetera:

  1. Para cargar los ingresos del mes ingrese CargarIngresos()
  2. Para cargar los egresos del mes ingrese CargarEgresos()
  3. Para mostrar el balance mensual de Ingresos y Egresos ingrese MostrarBalanceMensual()
  4. Para salir ingrese salir() en la consola
